# libshadertoy - uniforms example

This example shows how to use custom uniforms defined at runtime
with a custom shader, using libshadertoy.

## Dependencies

* libboost-filesystem-dev
* libglfw3-dev
* cmake
* git
* g++
* ca-certificates
* pkg-config

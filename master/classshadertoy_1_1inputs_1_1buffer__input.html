Detailed Description

Represents an input that uses a swap chain output as its source.

Note that setting min_filter to GL_*_MIPMAP_* will trigger automatic generation of mipmaps on every update to the source member of this input, which can impact performance of the resulting chain.

This class only holds a weak_ptr to the referenced input to prevent cyclic references. This means the target member must be owned by another object for as long as this input object exists.

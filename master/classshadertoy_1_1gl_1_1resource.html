Detailed Description

template<typename Final, typename Allocator, typename Error>
class shadertoy::gl::resource< Final, Allocator, Error >

Represents an OpenGL resource and manages its lifecycle. Uses move semantics, so only one Resource object can be the owner of a corresponding OpenGL resource, such as a texture, program, shader, etc.

◆ apply()

void shadertoy::draw_state::apply ( ) const

Apply the stored state to the current pipeline.

This will change the state of all supported glEnable features, as well as setting the equations and parameters for blending and depth processing. Clear settings will also be applied (color, depth, and stencil.) but the clear call will not be issued.

◆ member_output_t

typedef std::tuple<output_name_info_t, gl::texture *> shadertoy::members::member_output_t

Named output of a member.

This type holds the naming details of a given output in a render pass as well as a pointer to the texture holding the result of the pass.

◆ output_allocator_t

typedef std::function<std::tuple<rsize_ref, GLint>buffers::buffer_output &, const rsize_ref &)> shadertoy::members::output_allocator_t

Output allocator callback.

This function object returns for a given buffer_output discovered in a program, and a size reference of a swap chain member, the resulting size_reference object and OpenGL internal format of the texture that should be created.

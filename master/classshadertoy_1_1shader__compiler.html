Detailed Description

ShaderCompiler is a class for compiling shaders while maintaining file names and line numbers in error messages returned by the OpenGL GLSL compiler.

Load the sources in the provided shader object, and compiles the program. Any program building errors will be rewritten so known named sources will have their name in the output instead of numerical source ids.

Parameters
shaderThe shader
sourcesNamed sources to compile into the shader 
        Each pair of \p sources is a tuple (name, source) where name
        is the name of the shader source part, and source the actual
        GLSL code. The parts will be compiled in the same order as
        they are added to this vector.

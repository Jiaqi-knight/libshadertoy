Detailed Description

Represents a texture input to a buffer.

The default minification (resp. magnification) filter parameters default to GL_NEAREST.

◆ use_input()

virtual gl::texture* shadertoy::inputs::basic_input::use_input ( )
protectedpure virtual

Obtain this input's texture object.

This method must be implemented by derived classes in order to return a texture object to be used on the device. Note that an empty pointer can be returned, and will be replaced by an error texture for debugging.

Returns
Pointer to the texture object for this input. The returned pointer must have the same lifetime semantics as required by basic_input::use.

Implemented in shadertoy::inputs::buffer_input, shadertoy::inputs::image_input, shadertoy::inputs::checker_input, shadertoy::inputs::noise_input, and shadertoy::inputs::error_input.
